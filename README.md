# make

# make helps you create shapes from xml file in a second using only command line

# How to install make
$ cd ~

$ git clone https://github.com/veasnawt/make.git

$ cd make

$ ./install.sh

# How to use make

$ make.entershapehere
  
  Example: 
  $ make.rounded-shape
